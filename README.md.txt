# Django Demo App

This is a simple Django-based web application built for learning full-stack development concepts.  
The project contains APIs, database integration, and is fully deployable on Render.

---

##  Features

- Django backend
- REST API (optional based on your app)
- SQLite database
- Easily deployable on Render
- Clean folder structure

---

##  Folder Structure
project/
│── api/
│── project/        # (Your settings.py, wsgi.py folder)
│── db.sqlite3
│── manage.py
│── requirements.txt
